# Mini Study App
This mini study app is a java desktop application created using java swing. The app helps students study using simple study tools such as flashcards and a studytimer.

The project was built as a biginner-frendly java GUI project to practice:

- object oriented programming
- java swing
- event handling
- button and windows
- classes and inheritance

# Features
# flashcard
- add new flashcards
- save questions and answers
- open a separate windo to read saved flashcard
# study timer
- open a stopwatch
- help students track how long they studied for

# future improvemtents
- save flashcards permanently using file
- add multiple flashcards
- add quiz
